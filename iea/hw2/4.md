# (∀ g : G)[inv (inv g) ≟ g]
```
Seja g : G.
Calc:
  inv (inv g)
  = (inv (inv g)) * e [← G.idR]
  = (inv (inv g)) * ((inv g) * g) [← G.invL g]
  = ((inv (inv g)) * (inv g)) * g [← G.op-assoc]
  = eg [G.invL g:=(inv g)]
  = g [G.idL]
```

# [inv e ≟ e]
```
Calc:
  inv e
  = (inv e) * e [← G.idR]
  = e [G.invL e]
```

# (∀ g, g' : G)[inv (g * g') ≟ (inv g') * (inv g)]
```

Calc:
  inv (g * g')
  = (inv (g * g')) * e [← G.idR]
  = (inv (g * g')) * (g * (inv g)) [← G.invR g]

  = ((inv (g * g')) * (g * g')) * (inv g') * (inv g) [← G.op-assoc]
  = e * (inv g') * (inv g) [G.invL g:=(g * g')]
```
