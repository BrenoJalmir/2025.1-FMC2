# Demonstre a unicidade dos inversos.
```
-- (∀ g)(∃! g')[gg' = e & g'g = e]

Seja G um grupo.

-- (∀ g)(∃ g')[g'g = e & gg' = e]
Seja g : G.
Escolho inv g.
Imediato. [G.invL g; G.invR g]

-- (∀ g)(∀ i, i')[i, i' inversos de g ⇒ i = i']
Seja g, i, i' : G.
Seja h : i inverso de g ∧ i' inverso de g.
Calc:
  i
  = ie [← G.idR]
  = i(gi') [← h·r·r]
  = (ig)i' [← G.op-assoc]
  = ei' [h·l·l] 
  = i' [G.idL]
```
